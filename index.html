Bookmarklet to format the Toggl timer.

Download this project as a .zip file Download this project as a tar.gz file

How to use

Step.1

Drag & drop this button -> togglFormat into your bookmark bar.

Step.2

Access toggl timer.

Step.3

Click the "togglFormat" bookmarklet!
